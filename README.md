# tree-select
a tree-select component by react

##2017/02/26 更新 第一版

###当前功能：
  * 1.可展示树形列表
  * 2.可做多选操作，点击确定后会显示选中项
  * 3.可取消，取消会恢复上一次选中的状态
  * 4.各级菜单正确联动
